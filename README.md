🚀 **Built a Petrol Pump GUI Simulator using Python (Tkinter)** ⛽💻

Excited to share one of my recent projects — a **Petrol Pump Management Simulator** built using Python and Tkinter.

🔹 The application allows users to:
• Select Petrol or Diesel
• Enter fuel in Liters or Rupees
• Automatically calculate the total amount or fuel quantity
• Display results dynamically on a graphical fuel pump interface

🔹 Tech Used:
• Python
• Tkinter (GUI)
• Canvas & Dialog Boxes
• Basic Event Handling

✔ GUI Development
✔ User Interaction Handling
✔ Real-world problem modeling
