🚀 Built a Password Manager using Python & Tkinter
Today I built a Password Manager Desktop Application using Python and Tkinter GUI.
This project helps users securely store their website credentials and also generates strong passwords automatically.
🔧 Features of the Application:
• Generate strong random passwords
• Password strength indicator (Weak → Very Strong)
• Store credentials (Website, Email, Password) in a JSON database
• Automatic timestamp for when the password was saved
• Copy password directly to clipboard
• User-friendly GUI built with Tkinter
🛠 Technologies Used:
Python | Tkinter | JSON | Pyperclip
This project helped me understand:
• GUI development with Tkinter
• File handling using JSON
• Password generation logic
• Data storage and retrieval
